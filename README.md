# techyai_cj content system

The automation behind [@techyai_cj](https://www.tiktok.com/@techyai_cj), built with Claude.

- **docs/**: my landing page and free prompt guide, served with GitHub Pages
- **daily-sms/**: writes my daily content checklist every morning; my phone fetches it and texts it to me
- **carousel-maker/**: renders terminal style slide decks for TikTok photo posts and Instagram carousels
- **x-autoposter/**: generates long form AI news articles with branded header images

I make content about AI, Claude, and ChatGPT: news with a developer take, workflows you can copy, and things you didn't know AI could do.
